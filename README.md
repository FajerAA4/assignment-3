# My Professional Portfolio

## Project Description
This project is a personal portfolio website developed to showcase my academic work, technical projects, and professional profile.

The website demonstrates practical skills in:
- Front-end web development
- Responsive design
- API integration

The site includes dynamic features such as weather reporting, quote loading, project filtering, and personalization through saved preferences.

---

## Features

| Feature | Description |
|---------|-------------|
| Dark Mode | Saves user preference using `localStorage` |
| Time-Based Greeting | Displays greeting based on the system clock |
| Project Filtering | Filters projects by category and level |
| Project Sorting | Newest, oldest, alphabetical |
| Search Bar | Search projects by keyword |
| Quote API | Displays motivational quotes |
| Weather Widget | Shows live weather data |
| Name Memory | Saves visitor name in browser |
| Session Timer | Tracks active time on the site |
